Modify Custom Search Page

The custom search page is shown when 'Show all engines' is not checked and is intended to provide easy access to the favorite search engines. The search buttons can be positioned, colored and have their default status set.

Default search engines are marked with a thick border and will be searched when pressing Enter in the search query field. If there are more than one default engine, Just Search Portal will open multiple browser tabs with the search results.
